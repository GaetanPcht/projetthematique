## Prerequisites

- Python : [link](https://www.python.org/)

## Install

Git clone

```bash
git clone https://github.com/GaetanPcht/projetthematique.git
```

```bash
cd projetthematique
```

```bash
pip install django
```

```bash
pip install djangoframework
```

## Usage

Run this application

```bash
python manage.py runserver
```
